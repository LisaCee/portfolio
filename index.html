About

Hi. My name is Lisa Cee and I am a library worker by day, web developer by night. I work at one of the largest library systems in the nation and I have helped thousands of users interact with various websites. I see how people think the web should work and I see their frustrations when it doesn't. I design websites to make user experiences as straightforward as possible. I know that if a website is not easy and intuitive, then users will go elsewhere. My focus is on front-end coding, though I'm slowly expanding my knowlege to be full-stack. I want to know how it all works so that I can make it work better. I am a self-taught coder, making use of awesome resources like Khan Academy, Codecademy, Coursera and Free Code Camp. I love making responsive websites using HTML5, CSS3 and Javascript. All of my websites comply with W3C standards.
