About

Hi. I'm Lisa Cee; library worker by day, web developer by night. I am a self-taught coder based in Auburn, WA.

I work at one of the largest library systems in the nation and I help people who don't know how to use computers interact with websites. I see how people think sites should work and I see their frustrations when they don't.

I design websites to make user experiences as straightforward as possible. I know that if a website is not easy and intuitive, then users will go elsewhere. My focus is on front-end coding, though I'm expanding my knowlege to be full-stack. I want to know how it all works so that I can make it work better.
